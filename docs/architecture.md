# L0-L6 知识引擎架构

## 核心循环
L0原理 → L1实践 → L2偏差分析 → L3策略 → L4资产 → L5运行时 → L6风味映射

## 各层定义
- L0: 底层科学原理（物理/化学/生物），可证伪，含数值
- L1: 烹饪实践应用，每条L0对应N个L1
- L2: 实际操作偏差与失败模式
- L3: 纠偏策略
- L4: 可复用资产（配方、技法）
- L5: 运行时决策引擎
- L6: 风味感官映射
